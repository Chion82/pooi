# pooi
Permeable Online Objects Integration.

### Run
First, install Node.js(>= v4.0.0), Redis, MongoDB on your Linux/Macintosh.

__Development__

```bash
npm i
./node_modules/.bin/gulp
```

__Production__

Start.

```bash
npm start
```

Stop

```bash
npm stop
```

See pooi on http://127.0.0.1:6001.
